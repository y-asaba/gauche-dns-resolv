dns.resolv モジュールとは?

DNS サーバへの問い合わせを実行するライブラリになります。libresolv などの 外部のライブラリに依存せず、Gauche のみで記述されています。

ライセンス

BSD ライセンス

インストール

gauche-install と gauche-config を使ってインストールします。 $GAUCHDIR/share/site/lib にインストールする場合は、次のコマンドでインス トールします。 
$ gauche-install -T `gauche-config --sitelibdir` -m 644 dns/resolv.scm
他のディレクトリにインストールしたい場合は、-T の引数にインストールした いディレクトリを指定してください。

使い方

モジュールを使う場合は、use で宣言します。 
(use dns.resolv)

API

dns.resolv モジュール

サンプルプログラム

  • 名前からアドレスを調べる
  • アドレスから名前を調べる
  • MX レコードを調べる
